# Computer Network Chat App

A chat application developed as part of the Computer Network course project.

## Features

* Real-time client-server communication
* User login and registration
* Graphical User Interface (GUI)
* Web-based interface
* Message storage using SQLite database

## Technologies Used

* Python
* Socket Programming
* Tkinter
* SQLite
* HTML
* CSS
* JavaScript

## Project Structure

* `server.py` – Server application
* `client.py` – Client application
* `gui.py` – Graphical user interface
* `database.py` – Database operations
* `templates/` – HTML templates
* `static/` – CSS and JavaScript files

## Purpose

This project was developed to demonstrate client-server communication, networking concepts, and database integration in a chat application.
